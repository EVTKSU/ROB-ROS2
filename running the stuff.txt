before you do anything: 


in linux only:

Execute the commands below to setup the udev rules on Linux systems:

echo 'SUBSYSTEM=="usb", ATTRS{idVendor}=="03e7", MODE="0666"' | sudo tee /etc/udev/rules.d/80-movidius.rules
sudo udevadm control --reload-rules && sudo udevadm trigger

this sets up the USB connection requirements for the OAK D cameras with USB IO, since linux does not do that for you


1. make a venv in vscode.

1a. setup a venv
	cd Desktop/rob/ROB-ROS2
    sudo apt update

2. to rebuild a ws:

rm -rf build install log
source /opt/ros/jazzy/setup.bash
colcon build --symlink-install
source install/setup.bash

3. to use ethernet poe deployment:

manually set ipv4 to 169.254.1.223, subnet to 255.255.0.0

then use this line right before defining pipeline:

device_info = dai.DeviceInfo("169.254.1.222")

this will make the udp conn work!
