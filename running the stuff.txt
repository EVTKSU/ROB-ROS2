before you do anything: make a venv in vscode.

1a. setup a venv

    sudo apt update
	sudo apt install python3-venv
    python3 -m venv venv
	source venv/bin/activate
    pip install depthai numpy matplotlib ultralytics opencv-python pygame pyqt5 depthai-nodes

1b. to deactivate the venv:

	deactivate

1. to reactivate the venv:

	source venv/bin/activate



2. to rebuild a ws:

cd ~/Desktop/rob/ROB-ROS2
rm -rf build install log
source /opt/ros/jazzy/setup.bash
colcon build --symlink-install
source install/setup.bash

running:


to run auto:
---------------
ros2 launch minirob_auto minirob_auto.launch.py
---------------


to run teensy:
---------------
ros2 launch pi_teensy_udp pi_teensy_udp.launch.py
---------------


to run videorecord:
---------------
ros2 launch video_recorder video_recorder.launch.py
---------------


to run auto individual scripts

ros2 run minirob_auto minirob_auto lane_detection_node.py
